nRF24 MODULE CONNECTIONS
------------------------

nRF1:
  SCK  -> GPIO 14
  MOSI -> GPIO 13
  MISO -> GPIO 12
  CE   -> GPIO 26
  CSN  -> GPIO 15

nRF2:
  SCK  -> GPIO 14
  MOSI -> GPIO 13
  MISO -> GPIO 12
  CE   -> GPIO 27
  CSN  -> GPIO 25

nRF3:
  SCK  -> GPIO 18
  MOSI -> GPIO 23
  MISO -> GPIO 19
  CE   -> GPIO 4
  CSN  -> GPIO 2

nRF4:
  SCK  -> GPIO 18
  MOSI -> GPIO 23
  MISO -> GPIO 19
  CE   -> GPIO 16
  CSN  -> GPIO 17


OLED DISPLAY (I2C)
-----------------
SDA -> GPIO 21
SCL -> GPIO 22


BUTTONS
-------
LEFT   -> GPIO 33
RIGHT  -> GPIO 32
SELECT -> GPIO 5


POWER / ADC
-----------
Battery voltage divider -> GPIO 34 (ADC input only)

R1 = 100K
R2 = 100K

Battery + ---- R1 ----+----> ADC (GPIO 34)
                      |
                      R2
                      |
                    GND


NOTES
-----
- GPIO 34 is input-only (ADC safe).
